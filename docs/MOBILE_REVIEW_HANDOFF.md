## Feedback Space Template

Copy this into PRs, issue comments, and ChatGPT updates when asking Sergio to review.

```text
Mobile review link:

What I changed:

What to check first:

Your notes:
- Confusing:
- Too small / hard to tap:
- Not field-proof enough:
- Missing HVAC detail:
- Approved / not approved:
```

## Review Priorities

1. Can a tired HVAC tech understand the next step immediately?
2. Is the main button easy to tap from iPhone?
3. Does the route work as an end user would expect?
4. Are equipment docs and manual-match status clear?
5. Is the copy plain-language and professional?
6. Is it clear what is demo, planned, or live?

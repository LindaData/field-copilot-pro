# Agent Company Org Chart

## Top layer

```text
User
  owns final product direction, approvals, priorities, and taste.

CEO Commander Agent
  converts goals into task packets, routes work, resolves conflicts, and reports back.

Chief of Staff / Project Secretary
  keeps the logs, decisions, task state, open questions, links, and handoff records.
```

## Executive Council

```text
Chief Product Officer
  owns user experience, product scope, roadmap, and review-layer interpretation.

Chief Technology Officer
  owns architecture, repo strategy, implementation plan, and technical tradeoffs.

Chief Data Officer
  owns data sources, schemas, pipelines, analytics, documentation collection, and modeling readiness.

Chief Operations Officer
  owns task sequencing, checklists, delivery flow, and release readiness.

Chief Safety and Security Officer
  owns privacy, credentials hygiene, risk review, and sensitive workflow review.

Chief Quality Officer
  owns QA, testing, acceptance checks, mobile review, and regression review.

Chief Growth Officer
  owns SEO, positioning, onboarding, demo polish, and conversion paths.

Chief Financial Officer
  owns cost awareness, vendor tradeoffs, and cheap/free-first infrastructure choices.
```

## Communication rule

Specialists do not talk directly to the user unless explicitly assigned. They report to an executive, who reports to the CEO Commander, who gives the user a short answer.
